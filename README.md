# DailyVee.Tips

http://dailyvee.tips

Built by [Artemij Fedosejev](http://artemij.com) with React, Redux, Node, Bootstrap, SASS.

Uses Jest, Babel, Webpack, ESLint, stylelint, Yarn.

### Install

1. `yarn`
2. Create `config.json` file.

#### `config.json` file

```json
{
  "apiKey": "yourYokey"
}
```

+ `apiKey` - your YouTube API key.

### Run

`npm start`

### Test

`npm test`

### Add episode

`./addEpisode.sh`

### Add lesson

`./addLesson.sh`
